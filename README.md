# 📚 Rost

Приложение для учёта занятий логопеда-дефектолога. Позволяет вести базу воспитанников, записывать проведённые занятия и формировать отчёты.

## 🚀 Функционал

- 🔐 Авторизация (email + пароль)
- 👥 Управление воспитанниками (добавление, редактирование, удаление)
- 📅 Учёт проведённых занятий
- 📊 Отчёты и статистика (в разработке)
- 🔔 Уведомления о занятиях (в разработке)
- 🌙 Тёмная тема (в разработке)

## 🧱 Архитектура

Проект построен по принципам **Clean Architecture** с разделением на слои:

\`\`\`
lib/
├── core/
│   ├── theme/              # Темы приложения
│   ├── di/                 # Внедрение зависимостей (GetIt)
│   ├── utils/              # Утилиты и хелперы
│   └── extensions/         # Расширения (например, context.showMessage)
├── data/
│   ├── models/             # Модели данных (Child, Lesson)
│   └── repositories/       # Репозитории (Firebase, Hive)
├── features/
│   └── auth/
│       ├── bloc/           # BLoC (события, состояния, логика)
│       └── view/           # Экраны (логин, регистрация)
└── main.dart               # Точка входа
\`\`\`

## 🔧 Технологии

| Библиотека | Назначение |
|------------|------------|
| `flutter_bloc` | Управление состоянием |
| `firebase_auth` | Аутентификация |
| `cloud_firestore` | Облачная база данных |
| `hive_flutter` | Локальное кэширование |
| `get_it` | Внедрение зависимостей |
| `equatable` | Сравнение объектов |

## 📦 Запуск

1. Клонировать репозиторий:

\`\`\`bash
git clone https://github.com/Victor001Der/Rost.git
cd Rost
\`\`\`

2. Создать проект в [Firebase Console](https://console.firebase.google.com/) и добавить `google-services.json` в `android/app/`

3. Установить зависимости и запустить:

\`\`\`bash
flutter pub get
flutter run
\`\`\`
